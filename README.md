## 🚀 Key Sections & Architecture

* **About & Intro:** A clean, high-impact summary of my engineering manifesto and academic focus.
* **Featured Projects Grid:** Interactive showcase highlighting flagship projects (`VeriPolicy`, `openenv-code-review`, `air-sentinel`, `LabourLink`).
* **Experience & Technologies:** Structured overview of full-stack, AI, and DevOps skills.
* **Works Catalog (`myworks.html`):** A dedicated archive detailing all academic and personal engineering achievements.

---

## 💻 Tech Stack & Dependencies

* **Frontend Engine:** React 18 with TypeScript
* **Build Tooling:** Vite
* **Styling:** Tailwind CSS + custom glassmorphic CSS rules
* **Deployment & CI:** Connected to Vercel for continuous deployment on push

## ⚙️ Local Development Setup

To customize or run the portfolio site locally:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/nirnayyy/portfolio.git
   cd portfolio
   ```

2. **Install Package Dependencies:**
   ```bash
   npm install
   ```

3. **Launch Dev Environment:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` to view the local build.

4. **Compile Production Assets:**
   ```bash
   npm run build
   ```
